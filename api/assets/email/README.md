# Recursos d’e-mail (API de comandes)

- **`logo-white.png`** — logotip blanc sobre fons fosc, per a la capçalera del correu (adjunt `cid:lasaologo`).
- Es pot substituir per còpia d’un `logo_white-1920w.png` o de `public/images/logo-header-white.png` del projecte Angular (aquest repositori sol tenir còpia com a `logo-white.png` aquí).

Si s’esborra el fitxer, l’`api` omet l’`img` i mostra el nom de marca com a text.
